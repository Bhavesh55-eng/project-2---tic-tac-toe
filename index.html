How to Play

Game Rules:

  • Two players take turns marking empty cells.
  • The first to get three in a row (horizontal, vertical, diagonal) wins.
  • If all cells are filled without a winner, the game is a tie.

Controls:

  • Use mouse to click cells or buttons.
  • Use Tab to navigate interactive elements.
  • Use Arrow Keys to move focus across the game grid.
  • Press Enter or Space to select cells or activate buttons.
